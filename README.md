# Infrastructure Monitoring Stack

> Production-ready monitoring solution using Prometheus, Grafana, Node Exporter, Alertmanager and cAdvisor, fully deployed with Docker Compose.

## Architecture

*Coming soon — architecture diagram will be added as the stack is built.*

## Tech Stack

- **Prometheus** — Metrics collection and storage
- **Grafana** — Visualization and dashboards
- **Node Exporter** — Host-level system metrics
- **cAdvisor** — Container-level metrics
- **Alertmanager** — Alert routing and notifications
- **Docker Compose** — Service orchestration

## Quick Start

```bash
git clone https://github.com/YOUR_USERNAME/infrastructure-monitoring-stack.git
cd infrastructure-monitoring-stack
cp .env.example .env
# Edit .env with your own values
docker-compose up -d
```
